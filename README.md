# PalindromeCheckerApp
# Project Objective
This project demonstrates how to check whether a given string is a palindrome using basic Java concepts.
## Flow of Execution
1. Initialize input string.
2. Call recursive method:
   check(string, start, end)
3. Compare characters at:
   - start index
   - end index
4. If equal → Move inward
5. If not equal → Return false
6. Stop when:
   start >= end
7. Display result.

# Use Case 11 – Object-Oriented Palindrone Service

Description:

This use case applies object-oriented principles by
separating palindrome logic into a dedicated service
class (PalindromeService).

The main class handles:
- Input
- Service object creation
- Result display

The service class handles:
- Palindrome validation logic
