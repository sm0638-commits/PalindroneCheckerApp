# UseCase1PalindroneCheckerApp
## Objective
This is a Java project for a Palindrome Checker system.  
Currently, only the application entry and welcome message are implemented.
## Running the Program
Compile:
javac UseCase1PalindromeCheckerApp.java
Run:
java UseCase1PalindromeCheckerApp
## Current Flow
- Program starts from main() method
- Displays welcome message
- Shows version 1.0
- No palindrome logic yet
## Implemented Use Case
Use Case 1 – Application Entry & Welcome Message
